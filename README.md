# Escape The Underground
Escape The Underground is a simple platformer game where you accidentally fall under the ground and you need to find a way to escape. Prototype for HackClub Ember Jam 2026.

## Links:
- [Itch.io Page](https://crislzy.itch.io/etu)
- [Ember Jam Page](https://itch.io/jam/ember)

## Controls:
- Left  / Right arrow keys - Movement
- Spacebar - Jump
 
## Game Engine: 
- Game Maker
